# 玉龙雪山攻略 — 完整一日游指南

## 景区概览

玉龙雪山位于丽江市区北面约15公里，主峰扇子陡海拔5596米。核心游玩区域分四部分：

```
冰川公园（大索道 4506m→步行到4680m）
云杉坪（小索道 3200m）   ←进山→   蓝月谷（免费）
牦牛坪（中索道 3700m）
```

## 三条索道对比

| 索道 | 名称 | 海拔 | 票价 | 特点 |
|------|------|------|------|------|
| 大索道 | 冰川公园索道 | 4506m→4680m | ~140元 | 登顶看冰川，难度最大 |
| 中索道 | 牦牛坪索道 | ~3700m | ~65元 | 高山草甸看雪山全景 |
| 小索道 | 云杉坪索道 | ~3200m | ~60元 | 森林草甸，不缺氧 |

推荐顺序：大索道上冰川公园（上午）→ 蓝月谷（下午）。抢不到大索道可选云杉坪+蓝月谷组合。

## 抢票攻略

- 平台：微信小程序「丽江旅游集团」
- 时间：提前3天，每晚8点放票
- 例：6月7日去，6月4日晚8点抢
- 技巧：提前录好身份信息，选最早时段（8:00-9:00）
- 旺季极难抢到散票，推荐报纯玩一日团350-500元全包

## 一日团包含内容

索道票 + 进山费 + 氧气罐 + 羽绒服租赁 + 午餐 + 丽江古城接送。找团渠道：美团/飞猪/携程。

## 一日流程

| 时间 | 活动 |
|------|------|
| 6:30-7:00 | 出发/接人 |
| 7:30-8:00 | 进山交费 |
| 8:00-8:30 | 大索道上山（3356m→4506m） |
| 8:30-10:30 | 冰川公园，步行到4680m |
| 10:30-11:00 | 索道下山 |
| 11:00-12:30 | 蓝月谷 |
| 12:30-13:30 | 午餐 |
| 13:30-14:30 | 可选云杉坪 |
| 15:00-16:00 | 返程 |

## 必备物品

氧气罐（山下超市20-30元，山上80-100元）、防晒霜、墨镜、冲锋衣/薄外套、帽子、零食和水、充电宝、身份证

## 费用参考（自行前往）

进山费~100元 + 大索道~140元 + 氧气~25元 + 羽绒服~50元 + 交通~30元 ≈ 345-395元。跟团350-500元全包更省心。

## 高反提醒

大索道直达4506m，高反概率不低。慢慢走、不喝酒、睡好觉。心脏病/高血压不建议上大索道。严重头痛恶心立刻下山。葡萄糖口服液可救急。

## 其他注意事项

- 6月云南雨季，带伞
- 紫外线极强，防晒必需
- 建议穿舒适运动鞋
- 旺季索道票极难抢，提前规划
