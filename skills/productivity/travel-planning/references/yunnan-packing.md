# 云南旅行打包清单（6月雨季版）

适用于昆明、大理、丽江、香格里拉等云南目的地，6月出行。

## 气候要点

| 指标 | 数值 |
|------|------|
| 气温 | 18-26°C（白天） |
| 夜间最低 | 可降至 14°C |
| 天气 | 雨季，每天可能有阵雨 |
| 紫外线 | 高原地区，极强 |

## 衣物清单

### 核心（必带）

| 物品 | 数量 | 理由 |
|------|------|------|
| 半袖 | 3-4件 | 白天主力 |
| 长袖/衬衫 | 1-2件 | 傍晚降温时穿 |
| 薄外套/冲锋衣 | 1件 | 防晒服太薄，下雨+晚上扛不住 |
| 长裤 | 2条 | 白天一条、晚上一条 |
| 短裤 | 1条 | 白天逛古城可用 |
| 运动鞋 | 1双 | 每天暴走必备 |
| 拖鞋 | 1双 | 酒店+洱海边踩水 |

### 配件（必带）

| 物品 | 理由 |
|------|------|
| 折叠伞 | 6月雨季，每天可能一场阵雨 |
| 墨镜 | 高原紫外线，白天户外不戴睁不开眼 |
| 帽子 | 鸭舌帽/渔夫帽，防晒+防风 |
| 防晒霜 | SPF50+ |

### 可选

| 物品 | 理由 |
|------|------|
| 充电宝 | 古城/景区逛一天手机没电 |
| 学生证 | 很多景点半价 |
| 防水袋 | 洱海骑行/下雨时保护手机 |

## 常见错误

- ❌ 只带半袖不带外套 — 晚上+下雨会冷
- ❌ 不戴墨镜/帽子 — 高原紫外线比平原强很多
- ❌ 穿新鞋 — 每天暴走，新鞋磨脚
- ❌ 忘记伞 — 6月几乎每天一场阵雨
